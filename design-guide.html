Rainbow Stripe

Required on every page directly after the page header section. 14 spans (7 colors × 2), height 6px. Order: poppy → monarch → marigold → jade → cyan → plum → magenta, repeat.

Section Background Rhythm

Sections alternate between --cream and --cream2. Use --ink for a single dark accent section per page. Never use random hex values.

Grain Texture

Every page loads assets/grain.js via a deferred script tag. The grain overlay sits at z-index 9998 with mix-blend-mode: multiply.

07

Rules of the Road

Do
  • Use Outfit for all text — headings, body, labels, buttons
  • Set H1 at font-weight: 900
  • Set section H2s at font-weight: 700
  • Use var(--token) for every color reference
  • Square cards: border-radius: 0, 1.5px ink border, 5px offset shadow
  • Rotate card shadow colors with nth-child through the palette
  • Add a rainbow stripe after every page header
  • Use #fef5ec for page header backgrounds
  • Alternate sections: cream → cream2 → ink → cream
  • Pill shape only on Tour and Give nav buttons
  • Mark current page with class="active" on nav-item
  • Load assets/styles.css — never redefine :root tokens inline
Don't
  • Use Playfair Display, Georgia, or any serif font
  • Set section H2s at font-weight: 400 — it looks unfinished
  • Hardcode hex colors in page files (#aedceb, #236040, etc.)
  • Add a local :root block that overrides the shared token values
  • Round card corners — border-radius: 8px breaks the flat aesthetic
  • Add hover lift effects (transform: translateY) to cards or buttons
  • Use box-shadow with blur (0 8px 32px) — flat offset shadows only
  • Use pastel hardcoded backgrounds on cards (#fce8f0, #e6f4f0, etc.)
  • Skip the rainbow stripe on any page
  • Add additional Google Fonts families not already imported
  • Mix :root override colors that differ from styles.css
